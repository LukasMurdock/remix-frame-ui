# Getting Started

Remix Frame UI is a component library built for `remix/component` applications.

Use this guide track if you are new to the project.

## 5-minute path

1. Install packages from [Installation](#guide-installation)
2. Render your first page from [First page](#guide-first-page)
3. Submit a basic form from [First form](#guide-first-form)
4. Use the docs sidebar `Stable only` toggle while evaluating components

If you only want low-change APIs, stay on the stable primitives listed below.

## Package overview

| Package | Purpose |
| --- | --- |
| `@lukasmurdock/remix-ui-components` | Component API for `remix/component` apps |
| `@lukasmurdock/remix-ui-styles` | Layered CSS: tokens, primitives, patterns, components |
| `@lukasmurdock/remix-ui-core` | Low-level a11y and form helpers |

## Stable-first recommendation

Start with the stable primitives:

- `Form`
- `Flex`
- `Grid`
- `Layout`
- `Link`
- `Space`
- `Typography`

Then adopt experimental components as needed for product workflows.
