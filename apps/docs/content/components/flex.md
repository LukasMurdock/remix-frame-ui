# Flex

Maturity: stable

## When to use

- Renders a semantic `<div>` container for one-dimensional layout composition.
- Use `direction`, `align`, `justify`, `wrap`, and `gap` to compose toolbars, header actions, and inline control groups with consistent spacing.

## Import
